# Metadata

- Название: drawio-skill
- Автор: Agents365-ai; корпоративная адаптация: travinov
- Версия: 1.0.0
- Описание: Extension помогает GigaCode создавать и редактировать профессиональные диаграммы в формате `.drawio`, а также экспортировать их в PNG, SVG, PDF и JPG через установленный draw.io Desktop CLI. Поддерживаются архитектурные схемы, flowchart, ERD, UML, sequence/class diagrams, C4, сетевые и cloud-диаграммы, ML/DL-схемы, визуализация кода, Terraform, Kubernetes, docker-compose и SQL DDL.
- Применимость: Использовать, когда задача требует визуальной схемы с точной структурой, редактируемым `.drawio`-источником, корпоративными стилями, иконками draw.io, экспортом в изображение или PDF. В корпоративной среде macOS/Windows draw.io Desktop устанавливается через внутренний маркетплейс SberUserSoft: `https://sberusersoft.sigma.sbrf.ru/#search/Draw.io`. Если draw.io установлен нестандартно, путь задается через `DRAWIO_BIN` или config-файл `~/.drawio-skill/config.json` / `%USERPROFILE%\.drawio-skill\config.json`.
- Использование: Установить extension, распаковав ZIP в каталог skills агента: `mkdir -p ~/.agents/skills && unzip drawio-skill-corporate.zip -d ~/.agents/skills`. Затем установить draw.io Desktop через SberUserSoft, проверить CLI командой `drawio --version` или полным путем (`/Applications/draw.io.app/Contents/MacOS/draw.io`, `C:\Program Files\draw.io\draw.io.exe`). Для нестандартного пути создать config: `{"drawio_bin": "C:\\Program Files\\draw.io\\draw.io.exe"}`. Вызывать extension естественным запросом к GigaCode, например: "Нарисуй sequence diagram процесса оплаты в draw.io" или "Построй архитектурную схему сервиса и экспортируй в PNG".
- Эксперимент:* Работоспособность подтверждена статической проверкой состава extension, JSON-шаблона `config.example.json`, наличия инструкций `DRAWIO_BIN` / `config.json`, сборкой ZIP с корневой папкой `drawio-skill/`, проверкой `unzip -t` без ошибок и публикацией GitHub Release. Эффективность ожидается за счет явного корпоративного install path через SberUserSoft, поддержки macOS/Windows/WSL путей к `drawio`, сохранения оригинальных генераторов и валидаторов Agents365-ai, а также снижения ручных ошибок при поиске CLI в корпоративной среде.
